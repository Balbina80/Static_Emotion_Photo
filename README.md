## 🎯 Project Overview

This is a real-world client project developed for a professional photographer.

The goal was to create a clean, modern and emotionally engaging website that reflects the photographer’s style while maintaining a premium and minimal user experience.

The project focuses on strong visual storytelling, simplicity and elegant presentation of photography.

---

## ✨ Key Features

✔ Fully responsive design (mobile, tablet, desktop)
✔ Magazine-style photo gallery with categories
✔ Image lightbox with navigation
✔ Smooth hover effects and subtle animations
✔ Elegant UI with a soft, pastel colour palette
✔ Sticky navigation with branding
✔ Contact section with structured layout
✔ 🌍 Multilingual support (Polish / English)

---

## 🎨 Design Approach

The design is inspired by:

* minimalism
* soft pastel tones
* emotional storytelling through photography

The layout ensures that:

👉 photography remains the main focus
👉 the interface enhances, not distracts from the content
